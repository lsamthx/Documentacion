# EN

## aplicacionesEN.json

Este fragmento de código representa un objeto JSON que contiene una serie de descripciones de texto asociadas a diferentes claves. Es común en aplicaciones web y móviles usar este tipo de objetos para almacenar textos que se mostrarán en la interfaz de usuario, en este caso es para las traducciones de nuestra página web de Español e Inglés.

Lo mismo funciona para cada uno de los códigos mostrados en este apartado, pues representa cada una de las secciones principales de las cuales se desea su traducción correspondiente.
